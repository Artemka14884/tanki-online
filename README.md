# 🎯 Танчики Онлайн

Мультиплеерная игра «танчики» на Node.js + WebSocket. Играет один человек как
сервер (хост), остальные подключаются к нему по сети. Работает в браузере —
на компьютере и на телефоне.

## Что реализовано

- Огромная карта 3600×2400 с объектами: **стены** (непроходимые), **деревья**, **ящики** — у всех есть коллизия.
- Танки: **100 HP**, **медленное** движение (95 px/с).
- Снаряды: **быстрые** (780 px/с), урон **20 HP** за попадание (5 попаданий = смерть).
- **Возрождение через 3 секунды** в случайной точке карты, вне объектов.
- Управление: ПК — WASD/стрелки + мышь (прицел) + ЛКМ/пробел (выстрел).
  Телефон — два джойстика (движение и прицел) + кнопка «ОГОНЬ».
- Мини-карта, шкала здоровья, таблица очков (убийства/смерти), лента убийств.
- Сервер авторитетен: вся логика (движение, столкновения, урон) считается на сервере,
  чтобы никто не мог читерить на клиенте.

## Установка и запуск (хост / организатор игры)

Нужен установленный [Node.js](https://nodejs.org) (версия 16+).

```bash
cd tanks-online
npm install
npm start
```

В консоли появится:

```
Tanks Online сервер запущен!
Локально:     http://localhost:8080
Для друзей в одной сети: http://<твой-IP>:8080
```

Открой `http://localhost:8080` в браузере — попадёшь в игру сам.

## Как подключиться друзьям

### В одной Wi-Fi/локальной сети (проще всего)
1. Хост узнаёт свой локальный IP:
   - Windows: `ipconfig` (строка IPv4)
   - macOS/Linux: `ifconfig` или `ip a`
2. Друзья открывают в браузере (в т.ч. на телефоне, подключённом к той же Wi-Fi):
   `http://<IP-хоста>:8080`
3. На экране входа поле «Адрес сервера» уже будет заполнено правильным адресом
   автоматически — просто вводят имя и жмут «Играть».

### Через интернет (разные сети)
Нужно открыть/пробросить порт 8080 на роутере хоста (port forwarding на TCP 8080
в сторону локального IP хоста), либо использовать туннель, например:

```bash
npx localtunnel --port 8080
# или
ngrok http 8080
```

Полученную публичную ссылку (замени `http` на `ws` в поле адреса сервера у друзей,
если потребуется, например `wss://xxxx.ngrok-free.app`) друзья вставляют в поле
«Адрес сервера» на экране входа.

## Структура проекта

```
tanks-online/
├── server.js        # игровой сервер (Node.js + ws): карта, физика, коллизии, урон, респавн
├── package.json
└── public/
    └── index.html    # клиент: рендер на canvas, управление (клавиатура/мышь/тач-джойстики)
```

## Настройки (можно менять в server.js)

| Параметр                | Значение по умолчанию |
|--------------------------|-----------------------|
| Размер карты              | 3600 × 2400 px         |
| HP танка                  | 100                    |
| Урон снаряда               | 20                      |
| Скорость танка              | 95 px/с (медленно)      |
| Скорость снаряда            | 780 px/с (быстро)       |
| Время возрождения           | 3000 мс                 |
| Задержка между выстрелами    | 350 мс                  |

Все константы — в самом верху `server.js`, легко подкрутить под себя.
